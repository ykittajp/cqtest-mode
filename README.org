* cqtest-mode
This is contest logging major mode for Emacs.
This is *development* version.
It may not come when alpha version release...

** Install:
   1. Move Dirctory to path where load-path is set.
   2. Add to =init.el= :
: (require 'cqtest)
   3. Make the new file and =M-x cqtest-mode=.
